# Security Policy

Simple Reaper Reverb is a plain-text JSFX audio effect for REAPER. It does not use networking, shell commands, dynamic loading, or external dependencies.

## Reporting a vulnerability

If you find a security issue, please open a GitHub issue with:

- A description of the problem
- Steps to reproduce it
- Your REAPER version and operating system

For ordinary audio bugs, crashes, or unexpected sound, open a normal issue.
